✅ ЧТО ДОБАВЛЕНО:

🔑 МОДЕЛЬ ПОЛЬЗОВАТЕЛЕЙ:
• Таблица User с полями: id, username, password_hash, role, name, is_active
• Хеширование паролей с помощью Werkzeug
• Две роли: 'admin' (администратор) и 'seller' (продавец)
• Активные/неактивные пользователи

🛡️ СИСТЕМА БЕЗОПАСНОСТИ:
• Декораторы для проверки аутентификации:
  - @login_required - требует входа в систему
  - @admin_required - только для администраторов
  - @seller_or_admin_required - для продавцов и администраторов
• Сессии Flask для хранения данных пользователя
• Автоматическое перенаправление на страницу входа
