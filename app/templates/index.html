🎮 Онлайн‑магазин игр

Демо‑приложение: просмотр игр и проверка купленных игр по логину и паролю

REST API

Основные эндпоинты приложения:

  • GET /api/games — список игр (название, стоимость, жанр)
  • POST /api/user-games — купленная игра пользователя по логину и паролю

Тестовые пользователи (логин / пароль):

  • paimon / raidenei → игра с ID 1
  • tu134 / piupiupiu → игра с ID 2
  • kimkitsuragi / garrystop → игра с ID 3
