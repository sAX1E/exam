# 🏺 Приложение для аукционов антиквариата

Веб‑приложение для управления **участниками торгов**, **продавцами**, **лотами** и **аукционами**. Интерфейс предназначен для сотрудников площадки: администраторов и продавцов.

## 🚀 Возможности

### 🔐 Аутентификация и роли
- **Два типа доступа**: Администратор и Продавец
- **Безопасная авторизация**: хеширование паролей, сессии
- **Разграничение прав**: администратор управляет продавцами, продавец не видит этот раздел

### Основной функционал
- **Участники торгов**: добавление и просмотр (контакты/адрес)
- **Продавцы**: добавление и просмотр (только администратор)
- **Лоты**: добавление и просмотр (название, стартовая цена, описание, категория)
- **Аукционы**: создание и просмотр (дата, место/платформа, лот, продавец, статус, победитель/итоговая цена)

### Аналитика
- **Общая статистика** (кол-во участников/продавцов/лотов/аукционов)
- **Популярные категории** и **популярные лоты**
- **Аукционы по дате**, **лоты по категории**, **детали лота**

## 🛠 Технологии
- **Backend**: Python Flask
- **База данных**: SQLite (файл `data/auction.db`)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **ORM**: SQLAlchemy
- **Redis**: счётчики (посещения/успешные/неуспешные входы)

## 🚀 Установка и запуск

### Автоматическая установка (Windows)
1. Дважды кликните `install.bat`
2. Дождитесь установки зависимостей и инициализации БД
3. Откройте браузер: http://localhost:5000

### Ручной запуск

#### 1) Зависимости
```bash
pip install -r requirements.txt
```

#### 2) Инициализация базы данных (тестовые данные)
```bash
python app/init_db.py
```

#### 3) Запуск приложения
```bash
python app/app.py
```
