📊 МОДЕЛИ ДАННЫХ:

Bidder (Участники):
├── id (Primary Key)
├── name (Имя)
├── email (Email)
├── phone (Телефон, опционально)
└── address (Адрес, опционально)

Seller (Продавцы):
├── id (Primary Key)
└── name (Имя / название)

Lot (Лоты):
├── id (Primary Key)
├── name (Название)
├── starting_price (Стартовая цена)
├── description (Описание)
└── category (Категория)

Auction (Аукционы):
├── id (Primary Key)
├── date (Дата)
├── location (Место / платформа)
├── notes (Примечания)
├── status (active/completed/cancelled)
├── final_price (Итоговая цена, опционально)
├── lot_id (Foreign Key → Lot)
├── seller_id (Foreign Key → Seller)
└── winner_bidder_id (Foreign Key → Bidder, опционально)

Bid (Ставки):
├── id (Primary Key)
├── auction_id (Foreign Key → Auction)
├── bidder_id (Foreign Key → Bidder)
└── amount (Сумма)

🔗 СВЯЗИ:
• Seller 1:N Auction
• Lot 1:N Auction
• Bidder 1:N Bid
• Auction 1:N Bid
