# Схема базы данных «Аукцион»

## Информация
- Файл схемы: `current_schema.sql`
- Файл БД: `data/auction.db` (от корня проекта), в Docker: `/app/data/auction.db`
- Таблицы: user, bidder, seller, lot, auction, bid

## Использование
- Контроль версий структуры БД
- Восстановление схемы при необходимости
- Сравнение схем между окружениями

Таблицы также создаются автоматически при запуске приложения (Flask-SQLAlchemy `db.create_all()`).
