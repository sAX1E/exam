🏺 ПРИЛОЖЕНИЕ ДЛЯ АУКЦИОНОВ АНТИКВАРИАТА - ИНСТРУКЦИЯ ПО ЗАПУСКУ

📋 ЧТО СОЗДАНО:
✅ Полнофункциональное веб-приложение для аукционов антиквариата
✅ База данных SQLite с тестовыми данными (участники/продавцы/лоты/аукционы)
✅ Современный веб-интерфейс панели управления торгами

🚀 СПОСОБЫ ЗАПУСКА:

1️⃣ АВТОМАТИЧЕСКИЙ ЗАПУСК (РЕКОМЕНДУЕТСЯ):
   - Дважды кликните на файл "install.bat"
   - Дождитесь установки зависимостей и инициализации БД
   - Приложение автоматически запустится
   - Откройте браузер: http://localhost:5000

2️⃣ РУЧНОЙ ЗАПУСК:
   - Откройте командную строку в папке проекта
   - pip install -r requirements.txt
   - python app/init_db.py
   - python app/app.py
   - Откройте браузер: http://localhost:5000

3️⃣ БЫСТРЫЙ ЗАПУСК (если уже установлено):
   - init.bat - инициализация БД
   - run.bat - запуск приложения
